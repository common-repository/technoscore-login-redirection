== Description ==

Secure Frontend Ajax Login is  best plugin for WordPress to Login by ajax in popup and redirect to a specific page. 

= More details are as follows:- =
* It provides login by ajax in popup and redirect to particular page.
* It provides a Login button when it is clicked a simple login form will appear in popup.
* Just install the plugin and add the 'Techno Redirect' widget in the sidebar.

== Installation ==

1. Use WordPress' built-in installer
2. Download the ZIP file from WordPress website
3. Extract the technoscore-login-redirection directory to your computer
4. Upload the technoscore-login-redirection directory to the /wp-content/plugins/ directory
5. Activate the plugin in the Plugin dashboard
